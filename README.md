# 🐾 ARCHI'S SHELTER

A cute, strategic animal shelter management game starring **ARCHI PAPA** as the shelter mascot.

## Play
Open `index.html` in your browser, sign up with any username/password, and start managing your shelter!

## Features
- 🔐 **Multi-user accounts** — login & signup, progress saved per user (localStorage)
- 🐶🐱 **Dogs & cats** with personalities, ages, breeds, and unique stats
- 👥 **Visitor matching** — read profiles, find the perfect match for each visitor
- 💰 **Resource management** — balance income (adoptions, donations) vs expenses (food, vet, staff)
- 📈 **Progression** — level up to unlock cat room, vet clinic, play yard, volunteers, café
- ⏱️ **Day cycle** — 90-second days with morning/day/evening phases
- ⚡ **Pause & 2× speed** controls

## Controls
- Click an animal card to feed/play/train/treat
- Click a visitor to start an adoption match
- Auto-saves at end of each day and on close

Built with vanilla HTML5 + CSS + JavaScript. No dependencies, no backend.
